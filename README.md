## App Share
自分の作ったアプリを共有できるアプリ

[http://35.76.157.175/](http://35.76.157.175/)

![](public/image/app-share-top.png)

## 実装機能
- 投稿一覧・詳細・作成・編集・削除・並び替え(新着/人気)機能
- ページネーション機能
- いいね機能
- フォロー・フォロワー機能
- 新規登録・ログイン機能
- プロフィール作成・編集機能
- 通知機能
- コメント機能

## 使用技術
- Laravel
- jQuery(通知機能にajaxを使用)
- MySQL
- AWS(EC2, S3)
